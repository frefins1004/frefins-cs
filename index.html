    Firebase 설정
    아래 설정을 완료하면 실시간 데이터 저장이 활성화됩니다
    🔥 Firebase 프로젝트 설정
    1. Firebase Console에서 새 프로젝트 생성
    2. Authentication → 이메일/비밀번호 로그인 활성화
    3. Firestore Database 생성 (프로덕션 모드)
    4. 프로젝트 설정 → 앱 추가 → 웹 앱 → 설정값 복사 후 위에 입력
    📋 Trello 연동 설정
    Trello API Key: trello.com/power-ups/admin 에서 발급
    Board/List ID: Trello 카드 URL에서 확인 가능
    개선 요청, 오류 건이 자동으로 Trello 카드로 생성됩니다
    👥 사용자 계정 관리
    Firebase Authentication에 계정이 생성됩니다. 역할(role)에 따라 접근 가능한 메뉴가 달라집니다.
